Working on this assignment helped me finally understand what encapsulation actually does in a program. At first it was honestly confusing and I kept running into errors because I didn’t realize how strict Java is about keeping each class’s data private and only using the methods provided. Connecting the Engine, Car, Passenger and Train classes made it clearer why this matters and everything works more smoothly when each class controls its own information. The hardest part for me was matching the interfaces exactly, especially when tiny differences like boolean vs. Boolean caused big problems. I got stuck a lot in the beginning but going to TA really helped me slow down and understand what the interfaces were asking for instead of guessing. An advice i would give to someone taking this course is to test their code often, read the interfaces carefully and not get discouraged when they get errors because most of the time it’s just a small detail.
